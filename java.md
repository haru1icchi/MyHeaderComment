# Javaのコメント
javadocに準拠  

### ヘッダコメント
```java
/*
 * Title：
 * 説明 ：
 * @date Created on: 2015/12/24
 * @author Author: Haruyuki Ichino
 * @version 1.0
 *
 */
```

### クラスコメント
```java
/**
 *  このクラスはJavadoc説明用のクラスです。
 *  複数行に渡って記述することが可能です。
 *  @version 1.0
 *  @author Haruyuki Ichino
 *  @param int 年齢
 */
```
### 関数コメント
```java
/**
 * 関数の説明
 * @param hensuu_name 変数の説明 
 * @return 返り値の説明
 */
```
