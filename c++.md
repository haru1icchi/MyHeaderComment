### ヘッダコメント
```cpp
/*!
* @file fname.c
*
* @date Created on: 2015/12/24
* @author Author: Haruyuki Ichino
* @version 1.0
* @brief 説明文
* @par 追加見出し項目
*/
```

### 関数ヘッダ関係
```cpp
/*!
* @fn function_name
* @brief 概要
* @param (引数名) 引数の説明
* @param (引数名) 引数の説明
* @return 戻り値の説明
* @sa 参照すべき関数を書けばリンクが貼れる
* @detail 関数の詳細
*/
```
